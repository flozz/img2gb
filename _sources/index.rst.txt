Welcome to img2gb's documentation!
==================================

img2gb generates GameBoy Tilesets and Tilemaps from standard image (PNG,
JPEG,...). It converts the images into the GameBoy image format and generates
C code (``.c`` and ``.h`` files) that can be used in GameBoy projects.
